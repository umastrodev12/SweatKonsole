Recursos GitHub

O Emulador de Terminal feito em C Puro

Uma aplicação nativa Win32 inspirada no poder e visual do KDE Konsole, projetada para ser leve, modular e performática.

bash : konsole

=== Inicialização Concluída com Sucesso ===


you@konsole:[~]> make

Linkando arquivos e gerando konsole_gui.exe...

Compilacao concluida com sucesso!


you@konsole:[~]> ./sweatkonsole

Nativo & Veloz

Construído diretamente sobre a Win32 API e GDI do Windows. Sem frameworks pesados por trás, rodando em C puro.

Gerenciamento Estilo KDE

Prompt inteligente que identifica o diretório de trabalho atual dinamicamente e puxa o nome real do usuário do sistema.

Orquestrado por Init

Acompanha um fluxo modularizado onde um processo de inicialização dedicado gerencia o ciclo de vida do shell principal.
