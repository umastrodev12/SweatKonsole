# 🍦 SweatKonsole

A Terminal Emulator based in [KDE Konsole](https://github.com/KDE/Konsole).

SweatKonsole is a simple terminal emulator, we use **windows.h** to create the terminal window.

## Supported Plataforms

 - 🍎 MacOS
 - 🪟 Windows 11/10
 - 🐧 Linux

## SweatKonsole Directory Structure

| Directory      | Description                 |
| -------------- | --------------------------- |
| .vscode/       | VS Code config for project  |
| src/           | Source Code Of SweatKonsole |
